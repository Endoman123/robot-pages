A plugin that generates pages for FRC robot information.

== Description ==

This plugin creates a new post-type: Robot.

Robot pages hold info on the technical info about the robot, but also contain
info about the FRC season that the robot was used in.

== Installation ==

1. Download this plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. ...
